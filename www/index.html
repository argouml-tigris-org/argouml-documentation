Description

Welcome to the ArgoUML documentation project. Currently, this project is just a placeholder for the planned work on the ArgoUML documentation.

If you want to view the latest existing argouml documenation please use the 'nightly build' links on the left hand pane.

Currently the plan is to migrate all exisiting documentation translation projects into this project. At the moment, this plan is in the early stages.

Migration Aims

  • To reduce the workload of maintaining the documentation build process seperately for each language.
  • To improve the consistency between translations by having a common base manual.
  • To make it easier for contributors to set up new languages.

Migration Plan

  1. [LT] Create the argouml-documentation project (done)
  2. [LT?] Make an announcement to all the dev lists of all translation projects
  3. [LT/MVW] Grant roles in the argouml-documentation project for people from all different languages that want to work with the documentation.
  4. [DT] Move the current English documentation to the argouml-documentation project together with the tools.
  5. [DT] Update the Cookbook and website with the new instructions
  6. [DT(+LT?)] Update the build scripts to work with the new location (nightly build and release scripts).
  7. [???] Make a decision on how the structure of pictures for the separate languages are handled.
  8. [DT] Update the tools to be able to build multiple translations.
  9. [DT?] Migrate the German and Spanish translations into the argouml-documenation project

If you want to work on the ArgoUML project straight away, for now, please head back to the old location.

Project Goals (once set up)

  • Maintain up-to-date and relevant user documentation.
  • Translate the documentation into as many languages as possible.
  • Provide learning materials in the best possible format for users of ArgoUML

Principally, the user documentation consists of:

  • A User Manual, describing how to use ArgoUML and it's features.
  • A Quick Guide, describing how to get up and running with the basics.
